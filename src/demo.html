About the Widget

The OSGeo-Inject widget provides a lightweight way for OSGeo community projects to display their affiliation with the OSGeo Foundation.

Features:

  • Minimal footprint (~5KB combined)
  • Automatic light/dark theme detection
  • Configurable position and collapsed state
  • Dynamic announcements from OSGeo
  • Privacy-respecting analytics

Usage

Add the following to your HTML:

<link rel="stylesheet" href="https://affiliate.osgeo.org/css/osgeo-inject.css">
<script src="https://affiliate.osgeo.org/js/osgeo-inject.js" defer></script>

Configure with data attributes:

<script src="https://affiliate.osgeo.org/js/osgeo-inject.js"
        data-position="top-right"
        data-collapsed="false"
        data-theme="auto"
        defer></script>
